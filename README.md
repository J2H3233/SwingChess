# SwingChess

## 프로젝트 계획

> * 자바를 강의를 수강하면서 얻은 지식으로 간단한 스윙 프로그램 제작
    >   * 객체지향 프로그래밍을 준수하여 개발
>   * 기물의 좌표를 캡슐화
>   * 상속을 이용한 기물의 기능 구현
>   * 캡슐화를 이용한 정보 은닉
>   * 게임 작동의 추상화
> * MVC 디자인 패턴 활용
> * 객체 지향 5원칙(SOLID) 준수 노력
